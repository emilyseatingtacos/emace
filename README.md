# Daily Dashboard

A minimal privacy-friendly web app that greets you with a theme for the current day of the week, a motivational quote, and local weather.

## Features
- Daily themed background and quote
- Current date display
- Fetches local weather using [Open-Meteo](https://open-meteo.com/)
- No tracking or external analytics

## Usage
Open `index.html` in a web browser. Grant permission for location access to retrieve weather data.
