# emace

Brand assets for EGV Pet & Well Community Center.

## Signature logo
- **File:** `assets/egv-pet-well-community-center.svg`
- **Style:** continuous line illustration with a handwritten signature wordmark.
- **Colors:** charcoal ink (`#1c1c1c`) with a deep teal accent (`#0b8c6e`).
- **Usage tips:** the SVG scales cleanly for print and digital. Adjust the `--ink` and `--accent` CSS variables in the file to match alternate palettes.
